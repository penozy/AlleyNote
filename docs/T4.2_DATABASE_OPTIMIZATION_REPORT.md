## 🛠 新增功能

### BatchOperationService 批次操作服務

位置: `app/Infrastructure/Database/Services/BatchOperationService.php`

提供以下高效能批次操作方法：

- **`batchInsert()`** - 批次插入，預設批次大小 1000 筆
- **`batchUpdate()`** - 批次更新，預設批次大小 500 筆  
- **`batchDelete()`** - 批次刪除，預設批次大小 1000 筆
- **`batchUpsert()`** - 批次插入或更新，預設批次大小 800 筆

#### 使用範例

```php
use App\Infrastructure\Database\Services\BatchOperationService;

$batchService = new BatchOperationService();

// 批次插入 1000 筆活動記錄
$result = $batchService->batchInsert(
    'user_activity_logs', 
    ['uuid', 'user_id', 'action_type', 'action_category', 'created_at', 'occurred_at'], 
    $activityData, 
    1000
);

// 結果包含執行時間和效能統計
// $result = [
//     'inserted' => 1000,
//     'execution_time' => 0.0234,
//     'average_time_per_record' => 0.0234
// ]
```

## 📊 技術細節

### 索引策略說明

採用以下索引設計原則：

- **複合索引**: 針對多欄位 WHERE 條件設計
- **覆蓋索引**: 包含查詢所需的所有欄位
- **部分索引**: 針對特定條件的查詢（如 WHERE deleted_at IS NULL）

### SQLite 調整參數

```sql
PRAGMA foreign_keys = ON;          -- 啟用外鍵約束
PRAGMA journal_mode = WAL;         -- 啟用 WAL 模式提升併發
PRAGMA synchronous = NORMAL;       -- 平衡效能與資料安全
PRAGMA cache_size = -10000;        -- 10MB 記憶體快取
PRAGMA temp_store = MEMORY;        -- 暫存表存在記憶體
PRAGMA mmap_size = 268435456;      -- 256MB 記憶體映射
```
